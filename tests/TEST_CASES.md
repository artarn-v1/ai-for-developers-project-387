## Full E2E (`full-e2e.spec.ts`)

### TC-06: Полный цикл: Admin создаёт → Client бронирует → Admin подтверждает

| | |
|---|---|
| **Предусловие** | Seed-данные загружены |
| **Шаги** | 1. **API**: создать новый тип встречи 2. **UI**: открыть `/client/evgeny` — убедиться, что тип отображается 3. **API**: создать бронь на завтра 4. **UI**: открыть `/admin/evgeny-admin/meetings` — найти встречу, открыть детали, нажать «Confirm» 5. **API**: проверить, что `isConfirmed = true` |
| **Ожидаемый результат** | Тип создан и виден клиенту, бронь создана, админ подтвердил, статус изменился |
